# 📊 Finance Management Dashboard

A Streamlit-based finance dashboard that:
- Uploads bank statements (Excel)
- Cleans and categorizes transactions
- Shows income & expense distribution
- Tracks cumulative net income over time

## 🚀 Features
- Income & Expense Pie Charts
- Cumulative Net Income Line Chart
- Date range filtering
- Business-level categorization

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
